## Key Concepts

- **Stateful Environment**: Partial observability with hidden state
- **Investigation Tools**: Actions to gather evidence
- **Interventions**: Controls that unlock additional information
- **Trajectory Grading**: Scores based on process, not just outcomes
- **Adversarial Cases**: Replayable attack patterns
